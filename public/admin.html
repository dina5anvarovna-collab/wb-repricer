API-токен продавца WB

Откройте админку только по адресу сервера, например http://127.0.0.1:3001/admin.html — не двойным щелчком по файлу на диске (file://), иначе запросы к API не доходят и токен «не сохраняется». Если всё же открыли файл с диска — укажите URL сервера ниже и нажмите «Запомнить».

Нужен API-ключ категории «Цены и скидки» (личный кабинет продавца → Настройки → Доступ к API). Другие категории дадут ответ вроде «токен не поддерживается». Вставляйте ключ целиком, без кавычек. Сохранение в БД не ходит в WB; проверка — кнопками ниже, затем синхронизация.

Товары и минимальная цена

Колонка регионов заполняется после «Синк + мониторинг» (витрина). Выбранные регионы — вкладка «Регионы».

nmID Название Артикул Фикс./мин. ₽ Цена продавца ₽ Со скидкой ₽ Регионы: цена с СПП · % СПП Контроль

Импорт минимальных цен (CSV)

Формат: nmId;minPrice по одной строке. Заголовок можно пропустить.

Запуск проверок

Мониторинг: снимки цен (витрина + кабинет). Защита: расчёт и опционально выгрузка в WB.

Реальная выгрузка поднимает цену в кабинете WB. Убедитесь в min-правилах и регионе dest.
