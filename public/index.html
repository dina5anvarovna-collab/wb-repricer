WB — защита минимальной цены

Открыть админ-панель (русский интерфейс)

Кабинет WB + парсинг витрины (Кошелёк, регион dest), минимальная итоговая цена, автоподъём базы в WB.

REST API — порт из .env (по умолчанию 3001). Статус: /health

Быстрые ссылки

  • GET /api/settings/status
  • POST /api/settings/wb-token JSON {"token":"..."}
  • POST /api/catalog/sync
  • GET /api/catalog/products
  • GET /api/regions — справочник dest для региона
  • POST /api/fixed-prices/set JSON {"nmId":123,"targetPrice":999}
  • POST /api/monitor/run JSON {"maxProducts":5} (регион из REPRICER_WALLET_DEST)
  • POST /api/jobs/enforce-prices JSON {"dryRun":true,"dest":"-1257786"} — удержание цели; dryRun:false шлёт цену в кабинет WB
  • GET /api/monitor/snapshots
  • GET /api/cabinet-uploads — история выгрузок цен
